### `Quick Start`

git clone git@github.com:prinsapps/Student-Exercise-Monitor.git

cd Student-Exercise-Monitor

Frontend 
npm install
npm start
# client runs on http://localhost:8000

Backend
cd backend
npm install
npm start server
# Server runs on http://localhost:5000

Database Enviroment
change mongoos URI in .env file
